## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Dependencies

```
npm i --save @nestjs/config
npm i --save @nestjs/platform-fastify
npm i --save @nestjs/typeorm typeorm pg
npm i --save fastify point-of-view handlebars

npm i express-session
npm i -D @types/express-session
```
